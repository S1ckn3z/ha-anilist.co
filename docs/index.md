# AniList for Home Assistant — Documentation

## Overview

AniList for Home Assistant is a custom integration that connects your [AniList.co](https://anilist.co) anime and manga tracking data to your smart home dashboard. It provides sensor entities, calendar entities, and a custom Lovelace card with visual editor.
